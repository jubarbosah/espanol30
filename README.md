# Español 30 · Flashcards

App de flashcards (arquivo único, offline) para aprender espanhol peninsular partindo do português, com:

- **Repetição espaçada (SM-2)** com 4 botões (Errei / Difícil / Bom / Fácil)
- **Treino livre** sem limite (ES→PT e PT→ES)
- **Áudio** em espanhol (voz do aparelho)
- **Temas**, **Modo Armadilhas** (falsos cognatos) e baralho **🔀 Verbos que enganam**
- **Meus erros**: importe os erros do seu diário (JSON) como flashcards
- Tags 🟢 igual / 🟡 parecida com o português

Tudo roda no navegador, sem servidor. O progresso fica salvo no próprio aparelho.
